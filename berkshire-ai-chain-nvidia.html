NVIDIA 链
全球 · GPU 与 CUDA 生态垄断者
本页为「跨层全栈」框架示例：用公开年报 / 官网 / 新闻原创整理，不构成投资建议。
01芯片与算力层
NVIDIA 的核心壁垒是 GPU + CUDA 软件生态的双锁定。其数据中心 GPU（如 H100、Blackwell 架构的 B200 等）是当前大模型训练与推理的主流算力；CUDA 作为事实标准的编程框架，让全球开发者与科研工具深度绑定，形成「硬件 + 软件 + 开发者」的强网络效应。相比单纯卖芯片，NVIDIA 卖的是一整套难以替换的算力平台。
02云与基础设施层
NVIDIA 不直接运营公有云，而是与 微软、谷歌、亚马逊、Oracle 等云厂商深度合作，为其提供 GPU 集群；同时通过 DGX 整机、NVLink 高速互连、InfiniBand 网络（源自 Mellanox 收购）把单卡算力编织成「算力工厂」。它也从「卖芯片」延伸到「卖整套 AI 工厂方案」，吃下更多价值链。
03大模型与算法层
NVIDIA 自身不做消费级大模型，而是赋能所有大模型：其 GPU 是 GPT、Gemini、LLaMA 等基础模型训练的事实底座；并通过 NIM 微服务、NeMo 框架、TensorRT 推理优化等软件栈，把模型从「训练」到「部署」的环节都攥在手里。算法层越繁荣，对 NVIDIA 算力的需求越刚性。
04应用与终端层
算力向下游渗透到多个场景：游戏（传统基本盘，GeForce）、自动驾驶（车厂基于 NVIDIA 平台开发）、机器人（Isaac 平台）、工业数字孪生（Omniverse），以及医疗、科学计算等垂直领域。应用越广，算力需求越持续，形成需求端的长期拉力。
投资逻辑（框架示例）
护城河
CUDA 生态 + NVLink/网络 + 开发者习惯构成的三重壁垒，短期难被替代；先发优势在每一代架构（Ampere→Hopper→Blackwell）持续累积。
主要风险
① 大客户（云厂商、超大模型公司）自研 ASIC 芯片削减依赖；② 估值已反映极高成长预期，任何增速放缓都会被放大；③ 出口管制限制部分市场销售；④ 行业资本开支周期性强，可能存在产能过剩。
观察要点
数据中心营收占比与增速、 Blackwell 产能爬坡、软件（CUDA/NIM）订阅化进展、以及大客户自研芯片的替代节奏。
