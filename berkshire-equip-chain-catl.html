📖 人话解读

λ（lambda）=「跳变频率计」。把近 36 个月里单月涨跌≥15% 的月份数出来：λ↑ 是暴涨月数÷总月数（多头爆发力），λ↓ 是暴跌月数÷总月数（空头打击力），λ净 = λ↑−λ↓。

怎么读：λ净>0 且 总收益>0 才是真信号（跳变多 + 整体往上）；λ净>0 但总收益≤0 只是「坑里蹦」（死猫反弹），排除；λ净≤0 空头主导，不是我们的形态。

  • 宁德时代：λ净 +0.111、总收益 +71.8%，跳变偏多且整体向上 = 真信号。
  • 正面例：洛阳钼业 λ净 +0.222、总收益 +263.3%，跳变偏多又真涨 → ★泊松信号。
  • 反面例：古井贡酒 λ净 +0.028 看着多头，但总收益 -63.0% ≤ 0，属于「坑里蹦」，被排除。

记住：λ 只说「跳变多不多、偏哪边」，不说估值贵不贵、基本面扎不扎实。它是筛选工具，不是买卖信号，非投资建议。

公司 / 主题概述
宁德时代是全球动力电池与储能电池龙头，凭借麒麟 / 神行等电池技术与规模成本，绑定国内外主流车企与储能客户。盈利随动力电池装机与锂价波动，技术迭代与产能利用率是核心。
跨层全栈分析（5 层）
01上游 · 材料与部件
锂盐 / 正极 / 隔膜等外采 + 部分参股保供。关键材料外采 + 参股保供，保障供应链安全。
02中游 · 设备制造
电芯与电池包制造，技术迭代快（铁锂 / 三元）。电芯制造与快迭代技术，决定能量密度与成本。
03下游 · 电站与电网
动力电池配套车企 + 储能系统出货。深度绑定车企与储能客户，出货量规模领先。
04消费与场景
新能源车、电网侧 / 工商业储能。需求受新能源车与储能装机拉动，景气高。
05投资逻辑
护城河 = 技术 + 规模 + 客户绑定；风险 = 技术路线、价格战、锂价、竞争。技术规模 + 客户绑定构成壁垒，变量在技术路线与价格战。
