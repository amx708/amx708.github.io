公司 / 主题概述
Tesla Optimus 是全球人形机器人量产标杆，采用"车—机器人共享 AI 栈 + 自有工厂量产"策略。执行器自研、复用汽车制造与端到端神经网络，目标把机器人做成规模品而非展品。量产爬坡与单位成本是核心观察变量。（美股上市，无 A 股代码，财务详见 Tesla 年报）
跨层全栈分析（5 层）
01上游 · 核心零部件
自研旋转 / 线性执行器（电机 + 减速器 + 丝杠 + 传感），高度集成降本。
02中游 · 本体与执行器总成
本体在特斯拉工厂量产，复用汽车冲压 / 压铸 / 电驱 / 电池供应链。
03软件与具身智能
复用汽车端到端神经网络与 Dojo 训练，车—机器人共享 AI 栈。
04应用场景
先在特斯拉工厂（搬运 / 分拣 / 拧螺丝）跑通，再外溢更复杂场景。
05投资逻辑
护城河 = 制造迁移 + 数据闭环 + 自研；风险 = 量产爬坡、成本安全、竞争。

基于近36个月月收益，单月涨跌≥15%计一次跳变；λ净>0 且总收益>0 为入选信号。本指标仅反映价格跳变频率与方向，不替代估值与基本面，不构成投资建议。采集日 20260722。
📖 人话解读

λ（lambda）=「跳变频率计」。把近 36 个月里单月涨跌≥15% 的月份数出来：λ↑ 是暴涨月数÷总月数（多头爆发力），λ↓ 是暴跌月数÷总月数（空头打击力），λ净 = λ↑−λ↓。

怎么读：λ净>0 且 总收益>0 才是真信号（跳变多 + 整体往上）；λ净>0 但总收益≤0 只是「坑里蹦」（死猫反弹），排除；λ净≤0 空头主导，不是我们的形态。

  • Tesla：λ净 +0.111、总收益 +41.7%，跳变偏多且整体向上 = 真信号。
  • 正面例：洛阳钼业 λ净 +0.222、总收益 +263.3%，跳变偏多又真涨 → ★泊松信号。
  • 反面例：古井贡酒 λ净 +0.028 看着多头，但总收益 -63.0% ≤ 0，属于「坑里蹦」，被排除。

记住：λ 只说「跳变多不多、偏哪边」，不说估值贵不贵、基本面扎不扎实。它是筛选工具，不是买卖信号，非投资建议。
