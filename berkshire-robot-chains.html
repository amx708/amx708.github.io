资料说明 这份「机器人产业链地图」是怎么来的
本页是借鉴产业链分析方法、用公开资料自行整理的知识框架，把机器人按「层级」拆解，便于从投资视角理解其护城河与周期风险。
机器人是全球概念链（A股 / 港股 / 美股 / 非上市），跨市场 PE / PB 不可直接对比；A 股成分公司的财务 / 估值经真实年报与行情快照填充，无 A 股代码的全球标的（Tesla / NVIDIA / 优必选）保留占位诚实标注。
资料来源：各公司年报 / 官网 / 公开新闻 / 行业研报摘要——均为公开事实层；文中为框架级定性梳理，不编造具体财务数字，最新数据以公司公告为准。
跨层分析框架（5 层）
01
上游 · 核心零部件
减速器 / 丝杠 / 电机 / 传感器 / 视觉 / 控制器——机器人的"肌肉与感官"，价值量高、国产替代空间大
02
中游 · 本体与执行器总成
整机厂 / 关节模组 / 结构件——把零件拼成"身体"，量产与工艺是核心壁垒
03
软件与具身智能
AI 大脑 / 运动控制算法 / 仿真训练——机器人的"神经与思维"，大模型降低具身智能门槛
04
应用场景
工厂 / 商业服务 / 家庭 / 特种——落地的"用武之地"，先从结构化封闭场景跑通
05
投资逻辑
量产进度 / 订单定点 / 估值水位 / 护城河与风险——价值投资视角下的框架判断
产业周期判断标准
🔄 机器人产业链 产业周期怎么看

驱动变量：主机厂量产节奏（台数 / 良率 / 单位成本）、核心零部件订单与产能、政策支持（人形机器人 / 具身智能）、AI 大模型进展、估值水位（主题炒作 vs 业绩兑现）。

  • 量产爬坡：小批量内部部署 → 千台 → 万台，良率与成本曲线是关键；从演示视频到"经济量产"是最大鸿沟。
  • 订单与定点：零部件厂商获 Tier1 / 主机厂定点 → 收入验证；特斯拉 / Figure / 优必选 / 宇树等出货节奏是风向标。
  • 政策：国家与地方"具身智能 / 人形机器人"行动方案、首台套补贴、制造业转型升级，提供需求与信心支撑。
  • 技术：大模型（VLM / 端到端）降低具身智能门槛；执行器降本（丝杠 / 减速器国产化）决定放量速度。
  • 估值：主题阶段 PE / PB 失真，多看 PS / 订单 / 量产指引；警惕"故事"与业绩背离、筹码过热。
当前位置（定性框架，非数据结论）：机器人处于"0→1 量产导入期"：2025–2026 全球主机厂密集发布与小规模部署，核心零部件（减速器 / 丝杠 / 电机 / 传感 / 视觉）率先兑现订单；A 股以"主题 + 订单"驱动，估值含较高预期，须盯量产实绩与业绩兑现，警惕过热与单一客户依赖。
