公司 / 主题概述
优必选是港股"人形机器人第一股"，主打 Walker 系列人形机器人，聚焦教育、商用服务与工业场景。公司是国内少数具备全栈人形本体研发与量产能力的企业，但当前仍处高投入亏损期，需跟踪出货量、订单与降本进度。（港股上市，财务口径与 A 股不同，本环境无法经 A 股接口取数）
跨层全栈分析（5 层）
01上游 · 核心零部件
自研关节模组（电机 + 减速器 + 控制）构成上游核心部件能力。
02中游 · 本体与执行器总成
人形机器人本体厂，属中游"身体"制造与系统集成主体。
03软件与具身智能
具身智能算法（导航 / 交互 / 操作）是本体智能化核心。
04应用场景
教育 / 商用服务 / 工业为落地场景，先从结构化场景切入。
05投资逻辑
护城河 = 全栈本体 + 品牌先发；风险 = 持续亏损、出货 / 订单、人形竞争加剧。

🔄 本行业周期怎么看
机器人处于"0→1 量产导入期"：2025–2026 全球主机厂密集发布与小规模部署，核心零部件（减速器 / 丝杠 / 电机 / 传感 / 视觉）率先兑现订单；A 股以"主题 + 订单"驱动，估值含较高预期，须盯量产实绩与业绩兑现，警惕过热与单一客户依赖。
