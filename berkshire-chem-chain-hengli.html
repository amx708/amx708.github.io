基于近36个月月收益，单月涨跌≥15%计一次跳变；λ净>0 且总收益>0 为入选信号。本指标仅反映价格跳变频率与方向，不替代估值与基本面，不构成投资建议。采集日 20260722。
📖 人话解读

λ（lambda）=「跳变频率计」。把近 36 个月里单月涨跌≥15% 的月份数出来：λ↑ 是暴涨月数÷总月数（多头爆发力），λ↓ 是暴跌月数÷总月数（空头打击力），λ净 = λ↑−λ↓。

怎么读：λ净>0 且 总收益>0 才是真信号（跳变多 + 整体往上）；λ净>0 但总收益≤0 只是「坑里蹦」（死猫反弹），排除；λ净≤0 空头主导，不是我们的形态。

  • 恒力石化：λ净 +0.028、总收益 +17.0%，跳变偏多且整体向上 = 真信号。
  • 正面例：洛阳钼业 λ净 +0.222、总收益 +263.3%，跳变偏多又真涨 → ★泊松信号。
  • 反面例：古井贡酒 λ净 +0.028 看着多头，但总收益 -63.0% ≤ 0，属于「坑里蹦」，被排除。

记住：λ 只说「跳变多不多、偏哪边」，不说估值贵不贵、基本面扎不扎实。它是筛选工具，不是买卖信号，非投资建议。

公司 / 主题概述
恒力石化以「炼化—PTA—聚酯」一体化著称，大连长兴岛炼化项目打通全产业链，成本与效率领先。盈利受油价、PTA 与聚酯价差驱动，向下游新材料延伸。
跨层全栈分析（5 层）
01上游 · 原料
原油炼化供 PTA 原料，自给率高。炼化供 PTA 原料自给，锁定上游成本。
02中游 · 基础与炼化
炼化 + PTA + 聚酯纤维一体化。全产业链一体化提升效率与单吨成本优势。
03下游 · 精细与材料
涤纶长丝、薄膜、工程塑料。长丝薄膜塑料延伸，丰富产品组合。
04消费与场景
纺织服装、包装、电子材料。需求覆盖纺织包装电子，宏观敏感。
05投资逻辑
护城河 = 全产业链一体化 + 规模效率；风险 = 油价、价差、产能、需求。一体化规模效率构成壁垒，变量在油价与价差。

1）油价波动；2）PTA / 聚酯价差收窄；3）产能投放；4）需求疲软。

• PB分位线：PB历史分位 < 20% 视为破净/低PB安全边际区间 → 当前 PB分位 5.3% → 已触发
• 股息率线：现金牛安全边际≈股息率 ≥ 10Y国债(~1.6%,本站利率环境分位97.4%倒推) + 1.5%风险溢价 ≈ 3.1%；当前股息率 1.54% → 未达线（价格未给现金牛安全边际）
安全边际 = 价格显著低于内在价值的缓冲。本线为透明规则触发，非 DCF 精算；现金牛看股息率与低PB，成长看兑现与分位。不构成买卖建议。
治理 / 诚信维度（段永平透镜）
民企治理/诚信风险：重点警惕大股东占款、高比例股权质押、关联交易 tunneling；实控人变更、股权质押平仓、非标审计意见是高危信号。民企灵活激进，但诚信与透明度是核心变量——分红是否真实、财报是否被『调节』，需用经营性现金流与税费反推验证。
逆向压力测试 · 什么情景下永久损失资本（芒格 invert）
若以下情景同时或持续发生，恒力石化 的护城河与现金流将被实质性削弱：
  • 化工品价差长期磨底、产能出清不及预期
  • 原油/煤炭原料价格剧烈波动且无法转嫁
  • 安全环保事故导致大装置长期停车
  • 下游（地产/消费/汽车）需求疲软
  • 新增产能集中投放引发价格战
invert 思维：先想「怎么亏光」，再谈「怎么赚钱」。缓解因素通常已在护城河与财务板块体现（如低负债、长协、一体化）。
