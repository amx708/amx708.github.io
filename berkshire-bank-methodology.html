银行怎么读：核心指标框架

为什么银行不能用普通工商企业的估值框架？

银行是经营「风险」的企业：用存款人的钱去放贷，赚利差。它的「存货」是贷款（会违约），所以估值核心是资产质量与拨备厚度，而非单纯的营收增速。市场长期给银行PB<1（破净），本质是在给「看不见的坏账」打折。

1. 资产质量三件套：不良率 / 关注类 / 逾期

不良贷款率（NPL ratio）

不良余额 / 贷款总额。公认「已出问题」的贷款。但要警惕：不良是「分类结果」，银行有调节空间（借新还旧、展期可暂不计入不良）。

关注类贷款

虽未违约但存在隐患的贷款。关注类占比上升往往预示未来不良压力。看「（不良+关注）/ 贷款总额」比单看不良率更前瞻。

逾期贷款与「逾期偏离度」

逾期90天以上贷款原则上应计入不良。偏离度 = 逾期90天以上贷款 / 不良贷款，<100%说明分类严格（藏雷少），>100%说明有「逾期但没认定为不良」的粉饰嫌疑。

2. 拨备覆盖率：安全垫厚不厚

拨备覆盖率 = 贷款减值准备金 / 不良余额

越高越能吸收未来坏账。但过高的拨备也可能在「隐藏利润」（监管曾要求不低于150%，后改为120%-150%弹性区间）。拨备覆盖率高的银行（如招行390%+、杭州500%+）抗风险与利润释放空间都大。

拨贷比 = 贷款减值准备金 / 贷款总额

拨备覆盖率的补充，反映每100元贷款提了多少「保险」。与拨备覆盖率配合看，避免被低不良率+低拨备的「虚假安全」误导。

3. 净息差 NIM：银行盈利的发动机

净息差 = （利息收入-利息支出）/ 生息资产均值

银行最核心的盈利驱动。2022-2024年行业NIM从2.1%降至1.5%附近（LPR下调+存款定期化），是银行利润增速放缓的主因。常熟银行NIM 2.5%、邮储1.66%显著高于大行，源于存款成本优势。

4. 资本充足率：扩张的天花板

核心一级资本充足率 / 一级 / 资本充足率

监管红线（核心一级≥7.5%、资本充足率≥10.5%，系统重要性银行更高）。资本不足则无法放贷、无法分红。2025年财政部注资国有大行即补充核心一级资本。高ROE+高资本充足率=可持续扩张能力。

5. ROE 与 PB：破净之谜

为什么高ROE的银行PB却低于1？

招行ROE 13%、宁波13%，但PB仅0.8-0.9x；民生ROE 5%、PB 0.26x。市场给PB=1以下的折扣，是在为「潜在不良+让利政策+经济下行」定价。反过来，高ROE + 低PB + 高股息常被价值投资者视为「烟蒂变金条」——但前提是资产质量真实稳健，否则是价值陷阱。

实用口诀：看银行先看「不良偏离度+拨备覆盖率」定安全边际，再看「NIM+中收」定盈利趋势，最后用「PB分位+股息率」定买点。具体买点方法见估值温度计与三轮大底回测。
