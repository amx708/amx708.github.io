公司 / 主题概述
海信家电业务涵盖空调（含中央空调，日立 / 海信 / 约克多品牌）、冰箱、洗衣机，并通过收购日本三电（Sanden）切入新能源汽车热管理，打开第二曲线。背靠海信集团（视像、信芯等）在显示与芯片有协同。特点是「家电基本盘 + 汽车热管理期权」。
跨层全栈分析（5 层）
01上游 · 零部件与材料
压缩机、芯片（海信系协同）与三电热管理部件。
02中游 · 制造与品牌
空调 / 冰洗整机制造 + 中央空调（多联机领先）+ 汽车热管理部件。
03下游 · 渠道与出海
国内渠道 + 工程机 / 中央空调 + 海外；三电配套车企。
04消费与场景
家用空调冰洗、楼宇中央空调、新能源车热管理。
05投资逻辑
护城河 = 中央空调 + 三电协同；风险 = 地产、汽车周期、整合、价格战。
