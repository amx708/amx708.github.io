基于近36个月月收益，单月涨跌≥15%计一次跳变；λ净>0 且总收益>0 为入选信号。本指标仅反映价格跳变频率与方向，不替代估值与基本面，不构成投资建议。采集日 20260722。
📖 人话解读

λ（lambda）=「跳变频率计」。把近 36 个月里单月涨跌≥15% 的月份数出来：λ↑ 是暴涨月数÷总月数（多头爆发力），λ↓ 是暴跌月数÷总月数（空头打击力），λ净 = λ↑−λ↓。

怎么读：λ净>0 且 总收益>0 才是真信号（跳变多 + 整体往上）；λ净>0 但总收益≤0 只是「坑里蹦」（死猫反弹），排除；λ净≤0 空头主导，不是我们的形态。

  • 海信家电：λ净 +0.028、总收益 +22.3%，跳变偏多且整体向上 = 真信号。
  • 正面例：洛阳钼业 λ净 +0.222、总收益 +263.3%，跳变偏多又真涨 → ★泊松信号。
  • 反面例：古井贡酒 λ净 +0.028 看着多头，但总收益 -63.0% ≤ 0，属于「坑里蹦」，被排除。

记住：λ 只说「跳变多不多、偏哪边」，不说估值贵不贵、基本面扎不扎实。它是筛选工具，不是买卖信号，非投资建议。

公司 / 主题概述
海信家电业务涵盖空调（含中央空调，日立 / 海信 / 约克多品牌）、冰箱、洗衣机，并通过收购日本三电（Sanden）切入新能源汽车热管理，打开第二曲线。背靠海信集团（视像、信芯等）在显示与芯片有协同。特点是「家电基本盘 + 汽车热管理期权」。
跨层全栈分析（5 层）
01上游 · 零部件与材料
压缩机、芯片（海信系协同）与三电热管理部件。集团协同降采购与研发成本。
02中游 · 制造与品牌
空调 / 冰洗整机制造 + 中央空调（多联机领先）+ 汽车热管理部件。中央空调与热管理为增量。
03下游 · 渠道与出海
国内渠道 + 工程机 / 中央空调 + 海外，三电配套车企。工程与车企渠道绑定客户。
04消费与场景
家用空调冰洗、楼宇中央空调、新能源车热管理。热管理打开第二曲线。
05投资逻辑
护城河 = 中央空调 + 三电协同，风险 = 地产、汽车周期、整合、价格战。中央空调与热管理双线，但周期与整合是风险。

1）地产后周期；2）汽车景气与三电整合；3）价格战；4）原材料成本。

治理 / 诚信维度（段永平透镜）
地方国企治理风险：受地方国资委考核与区域财政影响，存在政绩导向投资、补贴依赖、资产注入/剥离等不确定性；关联交易与同业竞争需重点审视。相对民企，财务造假与实控人占款概率低，但经营效率与市场化激励偏弱。关注审计意见与独董独立性。
逆向压力测试 · 什么情景下永久损失资本（芒格 invert）
若以下情景同时或持续发生，海信家电 的护城河与现金流将被实质性削弱：
  • 地产后周期需求长期低迷，量价齐跌
  • 原材料（铜/钢/塑料）成本单边上行吞噬毛利
  • 海外贸易壁垒/关税使出海逻辑逆转
  • 价格战恶化致 ROE 永久性下台阶
  • 地产链大客户信用违约（应收账款坏账）
invert 思维：先想「怎么亏光」，再谈「怎么赚钱」。缓解因素通常已在护城河与财务板块体现（如低负债、长协、一体化）。

产业周期判断标准（家电产业链）
🔄 本行业周期怎么看
家电周期看“地产+补贴+出口+原材料”四角。当前阶段：地产竣工端仍偏弱，但以旧换新与存量更新托底内销；出口保持韧性，海外补库延续、新兴市场贡献增量；铜铝等原材料价格高位震荡，龙头用规模与套保对冲。行业整体处于“β弱但α强”的状态——龙头靠份额集中与高端化穿越周期，结构性机会大于总量机会。当前最值得跟踪：补贴延续性、外销补库节奏、原材料价格拐点、高端化渗透率能否继续提升。
