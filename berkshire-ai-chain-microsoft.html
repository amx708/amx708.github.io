📖 人话解读

λ（lambda）=「跳变频率计」。把近 36 个月里单月涨跌≥15% 的月份数出来：λ↑ 是暴涨月数÷总月数（多头爆发力），λ↓ 是暴跌月数÷总月数（空头打击力），λ净 = λ↑−λ↓。

怎么读：λ净>0 且 总收益>0 才是真信号（跳变多 + 整体往上）；λ净>0 但总收益≤0 只是「坑里蹦」（死猫反弹），排除；λ净≤0 空头主导，不是我们的形态。

  • Microsoft：λ净 +0.000 ≤ 0，空头跳变占上风，当前不是我们的关注形态。
  • 正面例：洛阳钼业 λ净 +0.222、总收益 +263.3%，跳变偏多又真涨 → ★泊松信号。
  • 反面例：古井贡酒 λ净 +0.028 看着多头，但总收益 -63.0% ≤ 0，属于「坑里蹦」，被排除。

记住：λ 只说「跳变多不多、偏哪边」，不说估值贵不贵、基本面扎不扎实。它是筛选工具，不是买卖信号，非投资建议。

跨层全栈分析（5 层）
01芯片与算力
Azure 是英伟达 H100/H200 全球最大采购方之一；同时自研 Maia 100 AI 加速器与 Cobalt CPU，并探索与 AMD、自研芯片的多供应商策略。数据中心覆盖全球 60+ 区域。
02云与基础设施
Azure 是微软 AI 收入的核心载体，提供 OpenAI 服务、Azure Machine Learning、AI Studio 等。企业用户可在 Azure 上微调、部署、管理大模型。AI 相关收入已连续多个季度贡献 Azure 增长的主要部分。
03大模型与算法
OpenAI 的 GPT-4 / GPT-4o / o1 系列是微软最重要的模型层。同时自研 Phi 系列小模型，并推出生成式 AI 安全工具（Azure AI Content Safety）。
04应用与终端
Microsoft 365 Copilot（办公）、GitHub Copilot（编程）、Dynamics 365 Copilot（CRM/ERP）、Power Platform Copilot（低代码）、Security Copilot（安全）。企业级 ARPU 提升是主要变现逻辑。
05投资逻辑
护城河 = 企业客户迁移成本 + OpenAI 合作关系 + 云规模。风险 = OpenAI 治理与竞合关系、AI 资本开支ROI、反垄断监管。微软是「AI 铲子+铲子柄」的典型案例：既卖算力，也卖应用。
